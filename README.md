# Para executar:
Rodar o commando "docker-compose up" na raiz projeto

O camando ira criar 3 containers
- clienteapi (API)
- sqlserver (BANCO DE DADOS)
- mssqltools (ferramentas de linha de comando do SQL Server para executar o script de criação de banco)

Após executar acessar o api no seguinte endereço: http://localhost:8081/swagger/index.html
